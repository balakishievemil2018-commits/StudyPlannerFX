### StudyPlannerFX — планировщик учебных задач

### Описание проекта
JavaFX-приложение для планирования и управления учебными задачами студентов.  
Позволяет создавать, редактировать, удалять и отслеживать выполнение заданий, а также смотреть простую статистику по учебной нагрузке.  
Подходит для студентов и преподавателей, чтобы планировать домашние задания, проекты и подготовку к экзаменам.

### Основные функции:
• CRUD-операции с задачами (создание, редактирование, удаление, просмотр)  
• Хранение данных о предмете, названии, дедлайне, приоритете и статусе задачи  
• Фильтрация задач по названию (и при необходимости по предмету)  
• Отдельное окно добавления/редактирования задачи  
• Мини-статистика: количество задач, количество выполненных, процент выполненных  
• При желании — отображение задач по датам (простое расписание)  
• Сохранение и загрузка данных в формате JSON  
• Валидация полей ввода и подтверждение удаления через диалоговые окна  
• Оформление интерфейса с помощью CSS (фон, цвета кнопок, шрифты, отступы)  
• Архитектура MVC и использование ObservableList для динамического обновления данных

### Установка и запуск
1. Убедитесь, что у вас установлен **Java JDK 17** или выше.  
2. Скачайте или клонируйте репозиторий:  
   `https://github.com/Emil/StudyPlannerFX.git`  
3. Откройте проект в **IntelliJ IDEA** или другой IDE с поддержкой JavaFX.  
4. Добавьте библиотеку **JavaFX** в зависимости проекта (через Gradle/Maven или настроив JavaFX SDK в настройках проекта).  
5. Убедитесь, что главный класс (например, `App.java` или `Main.java`) указан как точка входа.  
6. Запустите главный класс как **Java Application**.  
7. Приложение откроется, и вы сможете работать с учебными задачами.

### Примеры использования
• **Создание задачи:** нажмите кнопку «Добавить задачу», заполните поля (предмет, название, дедлайн, приоритет, статус) и нажмите «Сохранить».  
• **Редактирование задачи:** выберите задачу в списке и нажмите «Редактировать», измените поля и сохраните изменения.  
• **Удаление задачи:** выберите задачу в списке и нажмите «Удалить». Подтвердите действие во всплывающем диалоговом окне.  
• **Фильтрация задач:** введите текст в поле поиска — таблица автоматически отобразит только те задачи, название которых содержит введённую строку.  
• **Просмотр статистики:** перейдите на вкладку «Статистика» — будет показано общее количество задач, число выполненных и процент выполненных.

### Структура репозитория
src/  
├─ model/           # Модель данных (классы Task, Subject, статистика и т.п.)  
├─ view/            # FXML-файлы интерфейса и CSS-стили  
├─ controller/      # Контроллеры для управления логикой и обработкой событий  
└─ App.java / Main.java  # Главный класс приложения (точка входа)  

README.md           # Документация проекта  
testing/            # Документы по тестированию (план, отчёт, отзывы, issues)  
docs/               # Дополнительная документация (Agile, руководство пользователя и т.д.)

### Технические требования
• Язык: **Java 17+**  
• UI-фреймворк: **JavaFX SDK** (модули `javafx.controls`, `javafx.fxml`)  
• Архитектура: **MVC** (Model / View / Controller), использование `ObservableList`  
• IDE: **IntelliJ IDEA**, Eclipse или аналогичная  
• ОС: **Windows / macOS / Linux**  
• Дополнительно: библиотека **Gson** для работы с JSON-файлами
