# План тестирования StudyPlannerFX

## Цели тестирования
- Проверить, что операции с задачами (создание, редактирование, удаление, просмотр) работают правильно.
- Проверить поиск задач по названию.
- Проверить сохранение и загрузку задач в JSON-файл.
- Проверить обновление статистики (количество задач, выполненные, процент).

## Объект тестирования
Приложение **StudyPlannerFX** — JavaFX-приложение для планирования учебных задач.

Модули:
- вкладка «Задачи» (таблица, кнопки CRUD, поиск),
- вкладка «Статистика»,
- окно добавления/редактирования задачи,
- сохранение и загрузка данных (JSON).

## Среда тестирования
- ОС: Windows 10 / 11  
- Java: JDK 17  
- JavaFX  
- IDE: IntelliJ IDEA  
- Библиотека: Gson (для JSON)

## Основные сценарии
1. Создать новую задачу с корректными данными.
2. Попробовать создать задачу с пустым названием (должна сработать валидация).
3. Отредактировать существующую задачу.
4. Удалить задачу с подтверждением.
5. Отменить удаление задачи.
6. Найти задачу по части названия через поле поиска.
7. Сохранить текущий список задач в JSON-файл.
8. Загрузить задачи из сохранённого JSON-файла.
9. Проверить, что статистика обновляется после добавления/удаления задач.
10. Дать новому пользователю попробовать добавить и выполнить задачу без подсказок.
